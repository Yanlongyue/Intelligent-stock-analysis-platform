# 服务器文件排查指南

## ❌ 问题
```
python3: can't open file '/www/wwwroot/Intelligent-stock-analysis-platform/real_data_backend.py': [Errno 2] No such file or directory
```

## 🔍 排查步骤

### 1. 确认当前目录内容
```bash
cd /www/wwwroot/Intelligent-stock-analysis-platform
ls -la
```

### 2. 如果目录为空，需要重新拉取代码

```bash
# 检查Git仓库状态
cd /www/wwwroot/Intelligent-stock-analysis-platform
git status

# 如果显示 "not a git repository"，需要重新克隆
# 或者如果目录为空，也需要重新克隆
```

### 3. 重新拉取代码

```bash
# 进入上级目录
cd /www/wwwroot

# 备份（如果有需要保留的）
mv Intelligent-stock-analysis-platform Intelligent-stock-analysis-platform.bak.$(date +%Y%m%d)

# 从Gitee克隆（国内速度快）
git clone https://gitee.com/yourusername/Intelligent-stock-analysis-platform.git

# 或者从GitHub克隆
git clone https://github.com/yourusername/Intelligent-stock-analysis-platform.git

# 进入项目目录
cd Intelligent-stock-analysis-platform

# 确认文件存在
ls -la *.py
```

### 4. 安装依赖并启动

```bash
cd /www/wwwroot/Intelligent-stock-analysis-platform

# 安装Python依赖
pip3 install flask flask-cors akshare tushare

# 启动后端
nohup python3 real_data_backend.py > backend.log 2>&1 &

# 启动前端
nohup python3 -m http.server 8889 --bind 0.0.0.0 > frontend.log 2>&1 &

# 检查状态
netstat -tlnp | grep -E "8889|9000"
```
