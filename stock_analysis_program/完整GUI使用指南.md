## 📋 系统要求

### 软件要求
- Python 3.7+
- Tkinter（通常Python自带）
- 网络连接（用于获取股票数据）

### 硬件要求
- 内存：至少4GB
- 存储：至少500MB可用空间
- 显示器：分辨率1280x800或更高

## 🔧 首次运行配置

### 1. 安装依赖
如果首次运行失败，可能需要安装依赖：
```bash
pip install -r requirements_gui.txt
```

### 2. 配置Tushare Token
编辑配置文件：
```bash
vim config/tushare_config.py
```
将 `TUSHARE_TOKEN = "您的Tushare Pro Token请填写在这里"` 修改为您的实际Token。

### 3. 配置持仓
如果需要修改默认持仓，编辑：
```bash
vim config/position_config.py
```

### 2. 仪表盘标签页
- **持仓股票监控**：显示每只股票的实时价格和涨跌幅
- **统计分析**：显示总市值、总盈亏、风险等级等统计信息
- **操作按钮**：刷新数据、导出统计

### 3. 分析标签页
- **分析控制**：选择分析类型（完整/快速/持仓分析）
- **分析选项**：配置分析参数
- **分析进度**：显示分析进度和日志
- **操作按钮**：开始/停止分析、查看历史

### 4. 持仓标签页
- **持仓股票列表**：表格形式显示持仓详情
- **操作按钮**：添加、编辑、删除、刷新、导出

### 5. 报告标签页
- **报告列表**：按日期显示所有报告文件
- **报告预览**：预览选中的报告内容
- **操作按钮**：搜索、打开、删除、导出、打印

### 4. 常见问题解决

#### 问题1：无法连接到Tushare API
- 检查网络连接
- 验证Token是否正确
- 检查API服务状态

#### 问题2：GUI界面卡顿
- 减少自动刷新频率
- 关闭不必要的标签页
- 清理缓存文件

#### 问题3：报告无法预览
- 检查报告文件是否存在
- 检查文件权限
- 尝试重新生成报告

## 🔄 更新和升级

### 1. 检查更新
程序会检查是否有新版本可用。

### 2. 手动更新
```bash
# 备份当前配置
cp -r config/ config_backup/
cp -r data/ data_backup/

# 更新代码
git pull origin main

# 恢复配置
cp -r config_backup/* config/
cp -r data_backup/* data/
```

### 3. 数据迁移
如果需要迁移数据，可以导出数据库：
```bash
sqlite3 data/stock_gui.db .dump > backup.sql
```
