# 🎨 股票分析可视化界面架构设计

## 📋 总体架构

```
股票分析可视化界面
├── 主窗口 (MainWindow)
│   ├── 顶部菜单栏 (MenuBar)
│   ├── 主控制面板 (ControlPanel)
│   ├── 功能区选项卡 (TabView)
│   │   ├── 首页/仪表板 (Dashboard)
│   │   ├── 持仓管理 (PositionManagement)
│   │   ├── 分析报告 (ReportViewer)
│   │   ├── 图表展示 (ChartViewer)
│   │   ├── 系统设置 (Settings)
│   │   └── 帮助 (Help)
│   └── 状态栏 (StatusBar)
├── 系统托盘 (SystemTray)
├── 数据库 (SQLite)
└── 配置文件 (Config Files)
```

## 🔧 功能模块设计

### **1. 主控制面板 (ControlPanel)**
- **快速操作按钮**：
  - 一键运行完整分析
  - 快速查看持仓
  - 生成今日报告
  - 风险提示
  - 系统状态检查

- **实时信息显示**：
  - 当前持仓股票数量
  - 总市值
  - 今日盈亏
  - 风险等级指示器

### **2. 首页/仪表板 (Dashboard)**
- **KPI指标卡片**：
  - 总体仓位比例
  - 风险分布饼图
  - 近期收益趋势
  - 持仓股票概览

- **快捷操作区**：
  - 常用功能快速入口
  - 最近报告链接
  - 重要提醒通知

### **3. 持仓管理界面 (PositionManagement)**
- **持仓列表**：
  - 表格显示所有持仓股票
  - 支持添加/删除/编辑持仓
  - 实时价格更新
  - 持仓比例计算

- **持仓操作**：
  - 批量导入持仓
  - 导出持仓信息
  - 仓位调整建议
  - 风险重新评估

### **4. 分析报告查看器 (ReportViewer)**
- **报告列表**：
  - 按日期排序的历史报告
  - 支持搜索和筛选
  - 报告类型标识

- **报告预览**：
  - HTML报告内嵌浏览器显示
  - Markdown报告格式化显示
  - 支持报告下载和打印

### **5. 图表展示模块 (ChartViewer)**
- **持仓分布图**：
  - 饼图显示持仓比例
  - 柱状图显示风险分布
  - 趋势线显示价格变化

- **自定义图表**：
  - 支持多股票对比
  - 时间范围选择
  - 技术指标叠加

### **6. 系统托盘 (SystemTray)**
- **托盘图标**：
  - 实时显示系统状态
  - 右键快捷菜单
  - 通知提醒功能

- **后台监控**：
  - 价格预警提醒
  - 自动运行分析
  - 报告生成通知

## 🎯 用户交互流程

### **场景1：每日分析**
```
用户打开程序 → 显示仪表板 → 点击"一键分析" → 显示进度条 → 完成生成报告 → 自动打开报告查看器
```

### **场景2：持仓管理**
```
点击"持仓管理"标签 → 查看持仓列表 → 点击"添加持仓" → 输入股票代码和数量 → 保存更新配置
```

### **场景3：查看历史报告**
```
点击"报告查看器" → 选择日期 → 点击报告 → 在右侧预览区查看 → 支持下载或打印
```

### **场景4：监控预警**
```
系统托盘图标变色 → 右键点击 → 显示预警信息 → 点击查看详情 → 跳转到相应界面
```

## 📊 数据存储设计

### **1. SQLite数据库**
```
数据库: stock_analysis.db
├── 表: positions (持仓信息)
│   ├── id (主键)
│   ├── stock_code (股票代码)
│   ├── stock_name (股票名称)
│   ├── quantity (持仓数量)
│   ├── entry_price (买入价格)
│   ├── current_price (当前价格)
│   ├── stock_type (股票类型)
│   ├── risk_level (风险等级)
│   ├── target_position (目标仓位)
│   └── created_time (创建时间)
│
├── 表: reports (报告记录)
│   ├── id (主键)
│   ├── report_date (报告日期)
│   ├── report_type (报告类型)
│   ├── html_path (HTML文件路径)
│   ├── md_path (Markdown文件路径)
│   ├── summary (报告摘要)
│   └── created_time (创建时间)
│
└── 表: settings (系统设置)
    ├── id (主键)
    ├── setting_key (设置键)
    ├── setting_value (设置值)
    └── updated_time (更新时间)
```

### **2. 配置文件**
- `config/gui_settings.json` - GUI界面设置
- `config/user_preferences.json` - 用户偏好设置
- `config/color_theme.json` - 颜色主题配置

## 🎨 界面设计规范

### **颜色方案**
- **主色调**: 深蓝 (#2c3e50) + 科技蓝 (#3498db)
- **辅助色**: 绿色 (#27ae60) + 橙色 (#e67e22) + 红色 (#e74c3c)
- **背景色**: 白色 (#ffffff) + 浅灰 (#f5f7fa)

### **字体规范**
- **标题**: 微软雅黑 Bold, 16px
- **正文**: 微软雅黑 Regular, 12px
- **代码**: Consolas, 11px

### **布局规范**
- **主窗口**: 1200x800 像素
- **边距**: 统一10px边距
- **按钮尺寸**: 标准按钮 80x30px，大按钮 120x40px

## 🚀 技术实现要点

### **1. Python Tkinter**
- 使用ttk主题美化界面
- 使用ttk.Notebook实现标签页
- 自定义样式和主题

### **2. 多线程处理**
- 分析任务在后台线程运行
- 主界面保持响应
- 进度条实时更新

### **3. 图表渲染**
- 使用matplotlib生成静态图表
- 支持图表保存和导出
- 实时数据更新图表

### **4. HTML报告预览**
- 使用tkwebview或PyQt5的WebEngine
- 或使用内置http服务器+webbrowser

### **5. 数据库操作**
- 使用SQLite3
- ORM使用peewee或sqlite3直接操作
- 数据缓存和同步机制
