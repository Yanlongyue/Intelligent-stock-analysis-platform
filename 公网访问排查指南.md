## 🔍 排查步骤

### 1. 检查宝塔面板防火墙

```bash
# 查看宝塔防火墙状态
bt 13

# 或者直接在面板操作：
# 登录宝塔 → 安全 → 防火墙
# 确认放行：TCP 8889, TCP 9000
```

### 2. 检查系统防火墙

```bash
# 检查firewalld
systemctl status firewalld
firewall-cmd --list-ports

# 放行端口
firewall-cmd --permanent --add-port=8889/tcp
firewall-cmd --permanent --add-port=9000/tcp
firewall-cmd --reload

# 或者检查iptables
iptables -L -n | grep -E "8889|9000"
```

### 3. 检查阿里云安全组

登录 [阿里云ECS控制台](https://ecs.console.aliyun.com/)：
1. 找到实例 → 管理 → 本实例安全组
2. 配置规则 → 入方向
3. 添加规则：
   - 授权策略：允许
   - 协议类型：自定义TCP
   - 端口范围：8889/8889
   - 授权对象：0.0.0.0/0
   - 描述：股票分析平台前端
4. 同样添加9000端口

### 4. 测试公网访问

配置完成后测试：

```bash
# 在服务器上测试公网IP
curl -s http://101.133.150.164:9000/api/analyze_all | head -c 100

# 在本地电脑测试（Mac上执行）
curl http://101.133.150.164:8889/
curl http://101.133.150.164:9000/api/analyze_all
```

```bash
cp /www/wwwroot/Intelligent-stock-analysis-platform/stock_analysis_program/real_data_frontend.html \
   /www/wwwroot/Intelligent-stock-analysis-platform/index.html
```
